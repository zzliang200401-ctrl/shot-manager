# Shot Manager - Quick Start

## Download
https://github.com/zzliang200401-ctrl/shot-manager/archive/refs/heads/master.zip

## Prerequisites
- Node.js v20 (NOT v24)
  Download: https://nodejs.org/dist/v20.18.3/node-v20.18.3-x64.msi

## Install (One Time)
1. Unzip the file
2. Double-click: INSTALL.bat
3. Wait for installation to complete

## Run
Double-click: RUN.bat

Then open browser: http://localhost:3000

## Data
Your data is saved in: shotmanager.db

## Stop
Close the CMD window to stop the server
